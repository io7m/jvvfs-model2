A specification for the jvvfs package.

Requirements
------------------------------------------------------------------------

  Coq 8.4pl2

How to use
------------------------------------------------------------------------

  ./clean.sh
  ./build.sh

Source files and HTML documentation is produced and placed in the "doc"
subdirectory (which is created if it does not exist).
